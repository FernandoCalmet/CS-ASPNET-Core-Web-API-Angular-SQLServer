# CS ASPNET Core Web API with Angular & SQLServer

Project Sample using: Backend with ASP.NET Core Web API, Database with SQL Server, Frontend with Angular 12

Run Backend

```bash
dotnet run
```

Install Frontend Dependencies

```bash
npm install
```

Run Frontend

```bash
ng serve
```
